# CleanWave

CleanWave는 오염된 도심, 하천, 해안을 차례대로 정화해 나가는 2D 환경 정화 게임 프로젝트입니다.

## 프로젝트 개요

플레이어는 연결된 하나의 맵을 탐색하며 쓰레기를 수거하고, 올바른 수거함에 분리 배출해 구역의 정화율을 높입니다. 각 구역의 정화율을 100% 달성하면 다음 구역이 열리며, 수거 과정에서 얻은 코인으로 플레이어 장비를 업그레이드할 수 있습니다.

현재 저장소는 Unity 프로젝트 기준선과 기획 문서를 정리한 초기 MVP 개발 단계입니다.

## 기획 배경

CleanWave는 환경오염과 쓰레기 분리배출 문제를 게임 플레이 안에서 자연스럽게 이해하도록 만드는 것을 목표로 합니다. 퀴즈 중심 학습보다 직접 줍고, 분류하고, 정화율 변화를 확인하는 흐름을 통해 환경 회복의 과정을 짧고 명확하게 경험하게 합니다.

## 게임 목표

- 도심, 하천, 해안으로 이어지는 하나의 맵을 순서대로 정화합니다.
- 각 구역의 정화율을 100%까지 올려 다음 구역을 개방합니다.
- 점수, 코인, 정화율, 가방 상태를 확인하며 수거와 분리배출을 반복합니다.
- 코인으로 집게, 가방, 신발을 업그레이드해 수거 편의성을 높입니다.

## MVP 범위

- 연결형 1맵 구조: 도심 -> 하천 -> 해안
- 구역별 쓰레기 수: 도심 20개, 하천 25개, 해안 30개
- 쓰레기 수거와 수거함 분리 배출
- 정화율 기반 구역 개방
- 플레이어 장비 업그레이드: 집게, 가방, 신발
- HUD: 점수, 코인, 정화율, 가방
- 결과화면: 점수, 정화율, 별등급

## 프로젝트 열기

1. 저장소를 clone합니다.

```bash
git clone https://github.com/H0GUN3/Cleanwave.git
```

2. Unity Hub에서 `CleanWave/` 폴더를 프로젝트로 엽니다.
3. Unity Console에 오류가 없는지 확인합니다.
4. `Assets/Scenes/MainScene.unity`를 기준으로 작업을 이어갑니다.

## 프로젝트 구조

```text
CleanWave/
├── Assets/          # Unity 에셋, 씬, 스크립트
├── Packages/        # Unity 패키지 매니페스트
└── ProjectSettings/ # Unity 프로젝트 설정
```
