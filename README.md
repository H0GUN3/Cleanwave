# CleanWave

CleanWave는 오염된 도심, 하천, 해안을 차례대로 정화하는 Unity 2D 환경 정화 게임입니다. 플레이어는 연결된 하나의 맵을 탐색하며 쓰레기를 줍고, 올바른 수거함에 분리 배출해 구역 정화율을 올립니다. 각 구역을 정화하면 다음 구역이 열리고, 수거 과정에서 얻은 코인으로 플레이어 장비를 강화할 수 있습니다.

이 저장소는 CleanWave의 최종 Unity 프로젝트와 기획 문서를 포함합니다.

## 프로젝트 개요

CleanWave의 목표는 환경오염과 분리배출 문제를 직접적인 플레이 행동 안에서 자연스럽게 이해하도록 만드는 것입니다. 상시 퀴즈나 복잡한 학습 UI보다, 쓰레기를 직접 줍고 분류하고 정화율 변화를 확인하는 흐름을 중심에 둡니다.

게임은 `도심 -> 하천 -> 해안`으로 이어지는 연결형 1맵 구조입니다. 플레이어는 구역마다 배치된 쓰레기를 수거하고, 일반 수거함 또는 특수 수거함에 넣어 정화율을 올립니다. 정화율이 100%가 되면 다음 구역으로 향하는 길이 열립니다.

## 핵심 플레이 루프

1. 맵을 이동하며 쓰레기를 찾습니다.
2. `E` 키로 쓰레기를 주워 가방에 담습니다.
3. 수거함 앞에서 `E` 키로 현재 가방의 쓰레기를 분리 배출합니다.
4. 올바른 수거함이면 코인과 정화율이 증가합니다.
5. 잘못 분류하면 피드백을 통해 올바른 처리 방법을 확인합니다.
6. 구역 정화율 100% 달성 시 다음 구역이 개방됩니다.
7. 코인을 모아 상점에서 장비를 업그레이드합니다.
8. 모든 구역을 정화하면 결과 화면에서 최종 점수와 정화율을 확인합니다.

## 주요 기능

### 메인 메뉴

- `MainMenu` 씬에서 게임 시작과 종료를 선택합니다.
- 시작 버튼 클릭 시 효과음이 재생되고 `MainScene`으로 전환됩니다.
- 버튼 hover 시 전용 glow 이미지와 효과음이 표시됩니다.
- Editor Play 시작 시에도 메인 메뉴부터 시작하도록 구성되어 있습니다.

### 플레이어 이동

- `WASD` 이동을 지원합니다.
- `Shift`를 누르고 있는 동안 달리기 상태가 되며 이동속도가 30% 증가합니다.
- 장비 업그레이드로 증가한 기본 이동속도에도 달리기 배율이 적용됩니다.
- 이동 중 발소리 효과음이 재생되고 이동 속도에 따라 pitch가 조정됩니다.

### 쓰레기 수거와 분리 배출

- 쓰레기 근처에서 `E` 키를 누르면 가방에 담습니다.
- 가방에는 현재 담긴 쓰레기 아이콘이 표시됩니다.
- 수거함 앞에서 `E` 키를 누르면 현재 쓰레기를 배출합니다.
- 올바른 분리 배출 시 성공 효과음, 코인, 정화율 변화가 적용됩니다.
- 잘못 분리하면 정답 처리 방법을 알려주는 피드백이 표시됩니다.

### 구역 진행

- 맵은 `도심 -> 하천 -> 해안` 순서로 이어집니다.
- 각 구역의 정화율이 100%가 되면 다음 구역으로 가는 길이 열립니다.
- 구역 개방 시 중앙 팝업과 효과음이 표시됩니다.
- 구역마다 다른 배경 음악이 재생됩니다.

### 장비 업그레이드

- 플레이어 장비는 집게, 가방, 신발 중심으로 구성됩니다.
- 상점 근처에서 `F` 키를 누르면 업그레이드를 구매합니다.
- 업그레이드 성공 시 전용 효과음과 안내 메시지가 표시됩니다.
- 코인이 부족하면 구매 실패 메시지가 표시됩니다.

### UI와 결과 화면

- HUD는 점수, 코인, 정화율, 가방 상태를 중심으로 표시합니다.
- 오른쪽 상단 플레이 로그는 수거/분류 결과를 간단히 안내합니다.
- 퀴즈형 피드백은 구역 진행 중 필요한 순간에 표시됩니다.
- 결과 화면은 최종 점수와 정화율을 보여주고 다시하기/나가기 버튼을 제공합니다.

## 조작법

| 입력 | 동작 |
|---|---|
| `W`, `A`, `S`, `D` | 이동 |
| `Shift` 누르고 있기 | 달리기, 이동속도 30% 증가 |
| `E` | 쓰레기 줍기 / 수거함 분리 배출 |
| `F` | 상점 장비 업그레이드 구매 |
| `Q` | 가방 UI 표시 전환 |
| 마우스 | 메인 메뉴와 UI 버튼 선택 |

## 씬 구성

| 씬 | 설명 |
|---|---|
| `Assets/Scenes/MainMenu.unity` | 게임 시작 전 메인 메뉴 |
| `Assets/Scenes/MainScene.unity` | 도심, 하천, 해안이 연결된 메인 플레이 씬 |

Build Settings 기준으로 `MainMenu`가 0번, `MainScene`이 1번 씬입니다.

## MVP 범위

- 연결형 1맵 구조: 도심 -> 하천 -> 해안
- 쓰레기 수거와 수거함 분리 배출
- 구역 정화율과 정화율 100% 기반 구역 개방
- 코인과 플레이어 장비 업그레이드
- 구역별 BGM과 주요 효과음
- 메인 메뉴, HUD, 플레이 로그, 퀴즈형 피드백, 결과 화면

## 프로젝트 열기

1. 저장소를 clone합니다.

```bash
git clone https://github.com/H0GUN3/Cleanwave.git
```

2. Unity Hub에서 `CleanWave/` 폴더를 프로젝트로 엽니다.
3. Unity 버전은 `6000.3.16f1` 또는 Unity 6.3 LTS 계열을 사용합니다.
4. Unity Console에 compile error가 없는지 확인합니다.
5. `Assets/Scenes/MainMenu.unity`를 열고 Play를 눌러 메인 메뉴부터 확인합니다.

## 프로젝트 구조

```text
Cleanwave/
├── CleanWave/                         # Unity 프로젝트 루트
│   ├── Assets/
│   │   ├── Characters/Player/         # 플레이어 프리팹, 이동, 수거, 가방, 업그레이드 관련 스크립트
│   │   ├── Resources/                 # 런타임 로드용 UI, 폰트, 오디오 리소스
│   │   ├── Scenes/                    # MainMenu, MainScene
│   │   ├── Scripts/                   # Core, UI, Map, Trash, Upgrade, Player 런타임 로직
│   │   └── UI/MainMenu/               # 메인 메뉴 프리팹, 스크립트, 스프라이트
│   ├── Packages/                      # Unity 패키지 매니페스트
│   └── ProjectSettings/               # Unity 프로젝트 설정
├── CleanWave_프로젝트_계획서.md        # 확정 기획 기준 문서
├── docs/                              # 팀 가이드, 개발 규칙, 작업분해, 퀴즈/에셋 문서
└── README.md                          # 프로젝트 소개 문서
```
