# 역할별 작업 단위

이 문서는 시스템, 맵/Scene, 에셋/UI 담당이 무엇을 어디까지 작업하는지 정리한 실행용 작업 단위입니다.

## 시스템 담당

### 범위 포함

- 플레이어 이동 입력 처리
- 쓰레기 감지 및 수거 처리
- 가방 용량 제한
- 수거함 상호작용
- 쓰레기 종류별 올바른 수거함 판정
- 오분류 시 점수 `-5`
- 쓰레기 처리 시 코인 획득
- 정화율 계산: 제거한 오염도 / 전체 오염도 x 100
- 구역 정화율 100% 달성 시 다음 구역 개방
- HUD에 점수, 코인, 정화율, 가방 상태 전달

### 범위 제외

- `MainScene` 직접 배치 수정
- Tilemap 디자인 수정
- 최종 캐릭터/쓰레기/수거함 스프라이트 제작
- UI 디자인 이미지 제작
- 미니맵, 랭킹, 콤보, 배터리, 튜토리얼, Clean Cart 구현

### 단계

1. 준비: 스크립트 목록과 필요한 Prefab 목록 정리
2. 플레이어 이동: 2D 탑다운 이동, 충돌, 방향 상태 구현
3. 쓰레기 수거 + 가방: `F` 수거, 가방 5개 제한, 가방 내 쓰레기 선택 방식 정리
4. 수거함 분리수거: BinType 5종, TrashType 7종, 정답/오답 판정
5. 정화율/코인/구역 개방: 오염도 기준 정화율, Gate 개방
6. 업그레이드/HUD/결과화면: 장비 3종, HUD, 별등급 결과 연결

## 맵/Scene 담당

### 범위 포함

- `MainScene` 관리
- Unity Tilemap 구성
- 도심/하천/해안 구역당 20x15 Tile 배치
- 구역 간 연결 통로 구성
- 구역 잠금/개방 위치 구성
- 수거함 1세트와 상점 위치 배치
- 쓰레기 Prefab 배치
- 충돌 타일/장애물 배치
- 팀원들이 만든 Prefab, PNG, Tile을 Scene에 통합

### 범위 제외

- 시스템 로직 코드 작성
- 캐릭터 애니메이션 제작
- 쓰레기/수거함 원본 스프라이트 제작
- Git merge 최종 통합
- 복잡한 맵 장식 완성도 작업

### 단계

1. 맵 기준 준비: `MainScene`, 3구역 블록아웃, 연결 통로 초안
2. 도심 구역 완성 기준 만들기: 20개 쓰레기 공간, 수거함, 상점, 출구
3. 하천/해안 간소 구역 확장: 하천 25개, 해안 30개 배치 가능 공간
4. Prefab 통합과 배치: Player, Trash, Bin, Gate, Shop 배치
5. 충돌/정렬/카메라 점검: 이동 불가 영역, Sorting, Orthographic Size 확인
6. 최종 플레이 루트 고정: 발표 시연용 동선 안정화

## 에셋/UI 담당

### 범위 포함

- 플레이어 캐릭터 직접 제작
- 플레이어 Idle/Walk/Pickup 스프라이트 준비
- 쓰레기 7종 16x16 제작
- 수거함 5종 32x48 제작
- HUD 아이콘과 간단 UI 이미지 준비
- 무료 맵 타일/배경 에셋 후보 수집
- 파일명/폴더 규칙 준수

### 범위 제외

- `MainScene` 직접 배치
- 시스템 로직 코드 작성
- 맵 구조 설계
- 복잡한 UI 프로그래밍
- 모든 맵 장식 직접 제작

### 단계

1. 스타일 기준 확정: 참고 이미지, 무료 에셋팩 후보, 색감 기준 정리
2. 플레이어 캐릭터 제작: 32x48, down/up/side, Unity 테스트
3. 쓰레기 7종 제작: paper, can, plastic, vinyl, food, net, oil
4. 수거함 5종 제작: paper, can, plastic, general, special
5. UI 리소스 준비: 점수, 코인, 정화율, 가방, 별 아이콘
6. Unity 적용 점검: 크기, Pivot, PPU, 파일명 확인

## 파트 간 핵심 인터페이스

### 시스템 ↔ 맵

- Player 시작 위치
- Trash Prefab 필드
- Bin Prefab 타입
- Zone ID 또는 Gate 이름
- 구역별 쓰레기 수량

### 시스템 ↔ 에셋/UI

- Animator 방향명: `up`, `down`, `side`
- TrashType 이름
- BinType 이름
- HUD 아이콘 연결 이름
- Pickup 애니메이션 사용 여부

### 맵 ↔ 에셋/UI

- Tile 크기 32x32
- 캐릭터 32x48이 맵에서 어색하지 않은지
- 쓰레기 16x16 가독성
- 수거함 32x48 크기
- 무료 에셋팩 스타일 통일
